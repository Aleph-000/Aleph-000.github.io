这是这个个人博客的第一篇文章，也是“从 0 到 1 做实用好玩的个人博客”的起点记录。

我希望这里不只是一个静态名片，而是一处能长期维护的个人空间：介绍自己、展示项目、记录学习，也沉淀一些阶段性的想法。

当前主线 1 已经先完成基础框架：

- 支持用 Markdown 写博客文章，并由 GitHub 仓库存储
- 有个人介绍页面，展示身份、兴趣、经历、技能和获奖
- 有项目展示页面，整理正在做过的项目
- 通过 `https://aleph-000.github.io/` 访问
- 评论区暂时关闭，后续可以接入 Giscus 或全栈后端评论系统
